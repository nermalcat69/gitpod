# BitBucket

## Configuring Prebuild filters in the `.gitpod.yml`

There is no way to configure Prebuild filters in the `.gitpod.yml` for Bitbucket.

## Prebuild Webhooks

For a Bitbucket repositories, allow Gitpod to install repository webhooks, by granting `api` permissions in your [git provider integrations](https://gitpod.io/integrations)
